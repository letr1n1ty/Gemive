Gemive 設定

設定 Gemini API、介面語言、目標語言、字幕視窗、音訊與隱私。

介面

API

✓

API 金鑰只會存於這個瀏覽器。可用逗號輸入多組金鑰，啟動翻譯時會隨機選用其中一組。

語言

如果輸入語音已經是目標語言，仍然產生同語言口譯。一般建議關閉，避免背景聲或雜音被重複生成。

音訊

字幕視窗

開啟後，小視窗會上下分割：上半部顯示譯文，下半部顯示原文；關閉後譯文使用完整視窗高度。

啟用後，左上角 X 會變成 -；點擊 - 會收合成 Logo。短按 Logo 會展開，按住拖曳可移動位置。

字幕以時間窗口累積為主，句子邊界為輔；持續說話時舊文字不會立刻消失，超過可視高度後會自動滾動。

自動化

只會自動顯示字幕視窗，不會自動開始翻譯或擷取音訊。

每行一個網域，支援主網域與子網域，例如 youtube.com 會符合 www.youtube.com。

隱私

啟用後會先存進 Chrome 本機儲存空間。Chrome 擴充功能不能直接寫入任意系統目錄；下方目錄是匯出時在「下載」資料夾內使用的相對路徑。

除錯

除錯紀錄存在 Chrome 本機儲存空間的 gemive.debug.logs，API 金鑰會自動遮蔽。
